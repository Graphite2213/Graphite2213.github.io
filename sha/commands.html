Commands

Notice!

Commands that require no additional input (e.g. +help) are not listed here. To view all of the available commands use +help in the bot

CreateStand - Creates a new stand
Arguments: The stand's name.
Example: +createstand Star Platinum
   

Upgradehealth - Upgrades your health
Arguments: Amount of health  
Example: +upgradehealth 20
   

Rename - Renames your stand
Arguments: The new name
Example: +rename The World    


Use - Uses an item
Arguments: Item to use
Example: +use stand arrow    


Icon - Sets a new stand icon
Arguments: New icon URL
NOTICE: The URL must be a discord image url
Example: +icon link    


Description - Sets a new stand description
Arguments: The new description
Example: +description the same type of stand as star platinum  


Fight - Fights an user
Arguments: The user you'd like to fight
Example: +fight @DIO#5000    


Serverprefix - Sets the server prefix
Arguments: The new server prefix
Example: +serverprefix $    


Play - Plays music in a voice channel
Arguments: Song URL
Example: +play https://youtu.be/    


Urban - Searches the urban dictionary
Arguments: Term
Example: +urban jojo    


Transfer - Transfers balance to another user
Arguments: User, Money
Example: +transfer @Joseph#2020 200000¥
