Guide

Here's a basic explanation of the bot's commands:

/mintype: When the amount of people typing goes above this number it will trigger an automatic vote

/minvote: This is how many ✓ votes the vote needs to pass and create a new thread

/tname: Name of the thread created

/treason: Description (reason) of the thread created

/tarchive: Auto archive duration of the thread (in minutes)

/private: Sets the created threads to private

/cooldown: The cooldown before the bot calls another vote (in minutes)

For example: If you do /mintype 3, and 3 people start typing in a channel a vote prompt will pop up;


Lets also say you set /minvote 2 and 2 people vote for, but the third person votes X, a thread will be created. If it's the other way around a thread will not be created.


If hypothetically /private is on, and user1, user2 and user3 vote, but user4 watches the whole ordeal from a safe distance, a thread will be created but only user1-3 will be able to talk in it


In the case where a thread is created the thread will have the name you set using /tname and the auto archive duration you set using /tarchive


After a vote has passed or failed the bot will patiently wait in the background until it's cooldown set by /cooldown has passed. Then it will start listening for people typing again.


NOTE: /toggle DOES NOT disable the bot in its entirety, it's more of a permanent cooldown where the bot just wont be listening for typing users.
