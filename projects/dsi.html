Discord x Soundcloud integration

I started thinking about this project one day, the idea was there, so i created a concept and started working on it. Untilll i realized that Soundcloud API key applications were closed for years now. When they re-open i will definitely work on this project again (if i don't find another way to make it work that is).


My plan was to make a very simple app where the user would log into discord and soundcloud, then i would use the APIs of both of those and make a listening status on Discord for the user.
