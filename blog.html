Hello, in today's (very first!) blog post i will talk about this website and my projects and where all of this is going. So the P.S. from the welcome post still holds true, i still am confused on where this blog thing is going but i have a rough idea on what im going to be posting here and how often (once a week);

Ok to the next topic: Web development. It's pure pain.

It took me a month to build this website, and even then im not 100% satisfied on how it turned out. Im pretty sure that me doing it on my phone doesn't help a tiny bit but it was still extremely hard. Then again, im grateful that i got some experience in web development because now im overqualified for a web dev job. The memes online aren't even memes, i spent over 50% of my time centering divs and it destroyed my last 3 functioning braincells.

If you went through my website like once you would have noticed that different sections have different border colors, that's absolutely on purpose, i got a bright idea to color-code my projects and im doing the same thing with this blog. White border is general info or rambling or misc project updates (misc projects being everything that's not a Discord bot), a red border signifies AutoThreads updates/news and a blue border signifies SHA updates/news.

P.S. I didn't code the website on my phone because im an idiot or a maniac or both but because i don't have access to my computer right now. That's the same reason my bots aren't receiving updates and im not working on my projects.
